# Oficina: Inteligência Artificial para Devs 🤖💻

Este repositório contém o material prático utilizado no Trabalho de Extensão, voltado para alunos do ensino técnico em informática. O objetivo é demonstrar como utilizar assistentes de IA (como ChatGPT e Gemini) para acelerar o desenvolvimento, encontrar bugs e otimizar consultas em banco de dados, mantendo o raciocínio lógico e a ética profissional.

## Estrutura do Repositório

* `bug-fix.py`: Um script em Python contendo erros intencionais de sintaxe e lógica. O exercício consiste em criar o prompt correto para que a IA identifique, explique e corrija os problemas.
* `schema-sql.sql`: Um script PostgreSQL para criação de tabelas e inserção de dados fictícios. Os alunos utilizarão a IA para gerar consultas complexas (JOINs, agregações) a partir desta estrutura.

## Como utilizar na oficina
1. Faça um fork ou clone este repositório.
2. Abra os arquivos no seu editor de código preferido.
3. Siga o roteiro distribuído para aplicar os prompts nas ferramentas de IA.
4. Compare o código gerado pela IA com as boas práticas de programação discutidas em sala.
